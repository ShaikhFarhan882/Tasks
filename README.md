# Tasks
 Tasks is a android To-Do MVVM Architecture App written in Kotlin.(ViewModel, ROOM, Livedata, Coroutines)  
 
 # Features  
Create Tasks  
Update Tasks  
Delete Tasks  
Search Tasks  
Sort Tasks by Priority  
Material 3 Theme
